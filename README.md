# Java Tank Game 🎮

韩韩顺平 零基础30天学会Java里的坦克大战

## ✨ 更改点

-  增加敌方boss
-  可调节子弹数量、血量、敌方boss血量
-  可更改己方坦克速度
-  实时显示得分
-  显示敌方我方血量

## 🎮 操作说明

- **W** 向上移动
- **S** 向下移动
- **A** 向左移动
- **D** 向右移动
- **J** 发射子弹
- **P** 增加子弹数量
- **M** 减少子弹数量
- **H** 更改我发血量
- **E** 更改敌方血量
- **SPACE** 加速
- **ENTER** 减速

## 🧰 运行环境

- JDK 8+
- IntelliJ IDEA / Eclipse

## 🚀 运行方式

1. 克隆项目到本地
2. 使用 IDEA 打开项目
3. 运行主类即可启动游戏
